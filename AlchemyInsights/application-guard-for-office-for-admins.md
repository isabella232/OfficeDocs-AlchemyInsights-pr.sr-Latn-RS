# <a name="application-guard-for-office-for-admins"></a>Application Guard za Kancelarija za administacije

Application Guard za Kancelarija je dostupan organizacijama koje Microsoft 365 E5 ili Microsoft 365 E5 Mobility + Security licence. Korisnici u tim organizacijama moraju da koriste Microsoft 365 za preduzeća na trenutnom kanalu ili mesečnom kanalu za velika preduzeća.

**Naopako:** Application Guard za Kancelarija zahteva da Application Guard Microsoft zaštitnika (MDAG) prvo omogućen na uređaju. Za primenu MDAG uređaja na više uređaja, pogledajte priprema [za instaliranje aplikacije Application Guard Microsoft zaštitnika.](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-application-guard/install-md-app-guard)

Detaljne informacije o preduslovima, podešavanju, ograničenjima i rešavanju problema sa programom Application Guard za Kancelarija potražite u temi:

- [Faq: Application Guard za Kancelarija](https://support.microsoft.com/office/application-guard-for-office-9e0fb9c2-ffad-43bf-8ba3-78f785fdba46)
- [Podešavanje programnog zida Application Guard za Kancelarija](https://docs.microsoft.com/microsoft-365/security/office-365-security/install-app-guard)
